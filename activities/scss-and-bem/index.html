Learn SCSS & BEM

Example

Details

We are taking a step back and going from the basics of SCSS up to a scaleable SCSS architecture using most of the features available to us. We are also using the BEM naming methodology.

  • The goal for this activity is to refactor an existing vanilla CSS stylesheet into a SCSS stylesheet, or style architecture while implementing SCSS features.

Goals

  • Implement each of these SCSS features at least once
    • Variables
    • Nesting
    • Mixins
    • @if / @else conditionals
    • @for loops
    • Partials/Abstraction
    • Extending Styles
